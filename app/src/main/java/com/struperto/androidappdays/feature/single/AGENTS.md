# AGENTS

## Single

`Single` bleibt der Modusname und ist aktuell der ruhige, priorisierte `Bereichsfeed`.

## Regeln

1. `Single` ist lesend zuerst, nicht aktionsgetrieben.
2. Die Hauptstruktur ist ein priorisierter Gesamtfeed ueber Bereiche.
3. Jede Kachel ist ein klarer Output eines Bereichs.
4. Auf Mobile bevorzugt volle Breite je Kachel statt kleinformatiger Mehrspaltigkeit.
5. Die Oberflaeche bleibt ruhig, monolithisch und eher grau statt bunt und widgethaft.
6. `Single` lernt spaeter ueber Leseverhalten und Feedback fuer `Profil`; diese Logik darf die sichtbare Oberflaeche jetzt noch nicht verkomplizieren.
7. Verwende intern und in Texten `Bereichsfeed` statt `Bereichsinfo`.

## Nicht tun

- kein Dashboard mit vielen konkurrierenden Widgets
- keine Toolzentrale
- keine Schalterhaufen
- keine Aktionsebene, die den Lesefluss zerstoert
