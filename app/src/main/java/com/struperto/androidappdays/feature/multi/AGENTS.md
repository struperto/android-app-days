# AGENTS

## Multi

`Multi` ist aktuell eine sichtbare, aber nachrangige Produktschicht.

## Regeln

1. `Multi` nicht kuenstlich aufblasen, solange `Start` und `Single` nicht voll tragen.
2. In Doku und UI darf `Multi` als spaeterer Resonanzraum beschrieben werden.
3. Keine pseudo-magische, manipulative oder creepige Sprache verwenden.
4. Wenn du an `Multi` arbeitest, zuerst die Produktrolle schaerfen, dann erst die Runtime vertiefen.
5. Compile-safe Huelle ist okay; Fake-Tiefe ist nicht okay.
